Compile Instructions:
########################
- In order to compile this program, open a terminal and navigate to the directory containing the source files for the problem. 
Then, run the java compiler for the Prog1.java file

ex: javac Prog1.java


Run Instructions:
######################
- To run the application, use the following command:

Prog1 [-debug] [path to input file]

Where the input file is a file containing a list of 2D points for which the convex hull should be calculated, 
and -debug is an optional flag to specify additional output regarding calculating the convex hull.
