Compile Instructions:
########################
- In order to compile this program, open a terminal and navigate to the directory containing the source files for the problem. 
Then, run the java compiler for the Prog4.java file

ex: javac Prog4.java


Run Instructions:
######################
- To run the application, use the following command:

Prog4 [path to input file]

Where the input file is a file containing the number of posts along the river, and the cost matrix for the posts.
